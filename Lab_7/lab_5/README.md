# Lab 5 — TypeScript и проверка пропсов в React

## Описание работы
В данной лабораторной работе приложение на React было переписано с JavaScript на TypeScript.  
Были добавлены строгие типы для пропсов компонентов, состояния и обработчиков событий.

Работа состоит из двух частей:

- **Lab 5.1** — типизация компонентов и пропсов  
- **Lab 5.2** — типизация состояния и обработчиков событий  

---

## Использованные интерфейсы и типы

### User
Интерфейс описывает структуру пользователя:

- `name: string`
- `email: string`
- `age: number`

### Skill и SkillLevel
- `SkillLevel` — union-тип: `"Beginner" | "Intermediate" | "Expert"`
- `Skill` содержит:
  - `id: number`
  - `name: string`
  - `level: SkillLevel`

Все типы вынесены в отдельный файл **types.ts**.

---

## Почему TypeScript лучше PropTypes

- Проверка происходит **во время компиляции**, а не в браузере  
- Ошибки обнаруживаются **до запуска приложения**  
- Улучшается читаемость и поддержка кода  
- Исключено использование `any`
