# Lab 6.1 6.2

## Описание работы
В данной лабораторной работе приложение “Student Portal” было расширено для работы с динамическими маршрутами и query-параметрами.  
Использовались хуки **useParams**, **useLoaderData** и **useSearchParams**.  

Работа состоит из трёх частей:

- **Task 1** — динамический маршрут `/courses/:id`  
- **Task 2** — список курсов с переходом по ссылкам  
- **Task 3** — сортировка списка через query-параметр `?sort=asc|desc`  

---

## Использованные интерфейсы и типы

### Course
Интерфейс описывает структуру курса:

- `id: number` — уникальный идентификатор курса  
- `title: string` — название курса  
- `instructor: string` — имя преподавателя  
- `description: string` — описание курса  

Файл с данными — **data.ts**.

---

## Почему TypeScript удобен в React Router

- Проверка типов маршрутов и данных **во время компиляции**  
- Ошибки при неверных данных выявляются **до запуска приложения**  
- Улучшает читаемость и поддержку кода  
- Исключает ошибки при передаче данных между компонентами

- `useLoaderData()` — получает данные, загруженные через loader перед рендером

const { id } = useParams();
