## 2. Конкретные промпты

### Промпт 1:
"В чем разница между Flexbox в вебе (CSS) и в React Native? На что нужно обратить внимание?"

**Что получил:** Узнал ключевые отличия: в React Native `flexDirection` по умолчанию имеет значение `column` (в вебе — `row`), а все элементы по умолчанию имеют `display: flex`. Также нельзя использовать проценты для всех свойств подряд, часто выгоднее использовать `flex: 1`.

### Промпт 2:
"Как правильно обработать 'челку' на iPhone (Safe Area) в кастомном Header? Компонент SafeAreaView оставляет белые полосы."

**Что получил:** ИИ объяснил, что помимо `SafeAreaView` существует хук `useSafeAreaInsets` из библиотеки `react-native-safe-area-context`. Он позволяет точечно задать `paddingTop` для шапки, учитывая высоту системного бара, и при этом сохранить цвет фона (backgroundColor) единым для всей шапки.

### Промпт 3:
"Как в React Native реализовать Media Queries для адаптации под планшеты?"

**Что получил:** В React Native нет традиционных CSS Media Queries. ИИ показал использование хука `useWindowDimensions()`. Вытаскивая значение `width`, можно с помощью тернарных операторов менять стили на лету (например, `isTablet = width >= 768`).
